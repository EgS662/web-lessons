Заголовки

<h1> - заголовок 1 рівня.Цей заголовок найважливійший тому він виділений жирним шрифтом та має найбільший розмір серед усіх заголовків.

<h2> - заголовок 2 рівня.Цей заголовоку теж виділений жирним шрифтом але має меньший розмір ніж заголовок 1 рівня.

<h3> - заголовок 3 рівня.Цей заголовоку теж виділений жирним шрифтом але має меньший розмір ніж заголовок 1 та 2 рівнів.

<h4> - заголовок 4 рівня.Цей заголовоку теж виділений жирним шрифтом але має меньший розмір ніж заголовок 2 та 3 рівнів.

<h5> - заголовок 5 рівня.Цей заголовоку теж виділений жирним шрифтом але має меньший розмір ніж заголовок 3 та 4 рівнів.

<h6> - заголовок 6 рівня.Цей заголовоку теж виділений жирним шрифтом але він має найменьший розмір.

Теги для форматування тексту

<strong> - призначений для акцентування тексту. Виділяє текст жирним шрифтом

<em> - призначений для акцентування тексту.Виділяє текст курсивом

<small> - призначений для акцентування тексту.Зменшує розмір шрифту на одиницю в порівнянні зі звичайним текстом.

<ins> - призначений для виділення тексту, який був доданий до нової версії документа.Робить текст підкресленим.

<del> - використовується для виділення тексту, який був вилучений у новій версії документа.Робить текст закресленим.

<sub> - відображає шрифт у вигляді нижнього індексу. Текст при цьому розташовується нижче базової лінії інших символів рядка і зменшеного розміру.

<sup> - відображає шрифт у вигляді верхнього індексу. Текст при цьому відображається вище базової лінії тексту і зменшеного розміру.

Теги для розмітки тексту

<code> - призначений для відображення однієї або декількох рядків тексту, який являє собою програмний код.Текст при цьому відображається як моноширний текст зменшеного розміру.

<kbd> - використовується для позначення тексту, що набирається на клавіатурі або для назви клавіш.Текст при цьому відображається як моноширний текст.

<samp> - використовується для відображення тексту, який є результатом виведення комп'ютерної програми або скрипта.Текст при цьому відображається як моноширний текст.За допомогою стилів можна задати своє власне оформлення.

<pre> - визначає блок попередньо форматованого тексту. Такий текст відображається зазвичай моноширінним шрифтом і з усіма пробілами між словами.

<dfn> - застосовується для виділення таких термінів при їх першій появі в тексті.Такий текст відображається за допомогою курсиву.

<abbr> - вказує, що послідовність символів є абревіатурою.Такий текст в браузерах ніяк не виділяється, за винятком Opera до версії 15, яка додає до тексту пунктирне підкреслення.

Теги для форматування цитат

<q> - використовується для виділення в тексті цитат.Текст автоматично відображається в лапках.

<cite> - позначає текст як цитату або виноску на інший матеріал.Текст автоматично відображається курсивом.

<blockquote> - призначений для виділення довгих цитат.Текст відображається як вирівняний блок з відступами зліва і справа (приблизно по 40 пікселів), а також з відбиття зверху і знизу.

Теги для переносів тексту на інший рядок

<br> - цей тег робить переносить текст на інший рядок.

Теги для розмітки сторінки

<hr> - слугує для розмітки сторінки.Цей тег розміщує горизонтальну лінію на сторінці.

Теги для списків

<ul> - встановлює маркований список.

<ol> - встановлює нумерований список.

<li> - визначає окремий елемент списку.

Теги для адреси сторінки

<a href=“”> - задає адресу документа, на який слід перейти.

Теги для відображення зображення на сторінці

<img> - призначений для відображення на веб-сторінці зображень в графічному форматі GIF, JPEG або PNG.
